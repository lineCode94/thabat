# Product Requirements Document

# 06 - Gamification System

---

# Overview

The Gamification System is designed to increase user engagement, encourage long-term consistency, and celebrate personal progress.

The purpose is not to turn worship into a game.

Instead, it provides positive reinforcement that motivates users to continue their journey.

The system must never create unhealthy competition.

---

# Core Principles

The Gamification System rewards:

- Consistency
- Habit Building
- Long-Term Improvement
- Personal Progress

It should never reward showing off or competing with others.

Users compete only with their previous selves.

---

# XP (Experience Points)

Every completed worship activity grants XP.

XP is used to:

- Unlock Titles
- Unlock Badges
- Increase User Level

XP never decreases.

Historical XP is always preserved.

---

# XP Calculation

Each Worship Item has its own XP value.

Example

Fajr Prayer

20 XP

Morning Azkar

10 XP

Quran Reading

2 XP per Page

Administrators can modify XP values.

Changing XP affects future records only.

Historical XP remains unchanged.

---

# User Level

The application has a Gamification Level.

This is different from the Worship Level.

Example

Worship Level

Level 3

Gamification Level

Level 18

The Worship Level is controlled by the Mentor.

The Gamification Level is calculated automatically from XP.

---

# Consistency Index

Consistency is the most important metric.

It measures how consistently the user completes assigned worship.

Range

0%

to

100%

Consistency should have a higher importance than XP.

---

# Streak

A Streak counts consecutive days with activity.

Missing a day resets the streak.

Example

7 Days

15 Days

30 Days

100 Days

The current streak and longest streak should both be displayed.

---

# Achievements

Achievements celebrate milestones.

Examples

- First Week Completed
- First Month Completed
- 100 Prayers
- 500 Quran Pages
- 50 Morning Azkar
- 10 Lessons Attended
- First Promotion

Achievements are unlocked automatically.

---

# Badges

Badges are visual rewards.

Examples

- Consistent
- Quran Lover
- Early Bird
- Knowledge Seeker
- Helpful Mentor
- Dedicated

Users may showcase earned badges on their profile.

---

# Titles

Titles represent long-term progress.

Examples

- Beginner
- Growing
- Consistent
- Dedicated
- Inspiring

Titles are automatically assigned.

---

# Heatmap

Every user has a contribution-style calendar.

Each day displays:

Green

Excellent

Yellow

Partial

Gray

No Activity

Locked weeks display a lock icon.

The Heatmap should encourage users to maintain consistency.

---

# Journey Page

Every user has a personal Journey.

The Journey includes:

- Current Worship Level
- Current Gamification Level
- XP
- Consistency
- Current Streak
- Longest Streak
- Achievements
- Badges
- Weekly Reviews
- Heatmap
- Progress Charts

---

# Weekly Missions

Mentors and Administrators may assign Weekly Missions.

Examples

- Read 40 Quran Pages
- Memorize One Surah
- Read One Useful Book
- Attend Two Lessons
- Visit a Relative

Mission completion grants bonus XP.

---

# Encouragement Messages

Whenever a milestone is reached, the user receives a positive message.

Examples

"Excellent! Seven days of consistency."

"You are building a great habit."

"Keep going. Every step matters."

Messages should always encourage.

---

# No Public Rankings

The application must never display public leaderboards.

Users should never compare themselves with others.

The focus is always personal improvement.

---

# Mentor Recognition

Mentors receive recognition for supporting users.

Examples

- 10 Successful Promotions
- 100 Weekly Reviews
- 50 Active Users

This encourages mentorship quality.

---

# Future Features

Future versions may include:

- Seasonal Challenges
- Community Campaigns
- AI Motivation
- Personalized Recommendations

These are not part of Version 1.

---

# Acceptance Criteria

✓ XP is awarded automatically.

✓ Consistency is calculated automatically.

✓ Streak is updated daily.

✓ Achievements unlock automatically.

✓ Badges unlock automatically.

✓ Titles update automatically.

✓ Weekly Missions grant bonus XP.

✓ Heatmap displays daily activity.

✓ No public leaderboards.

✓ The experience remains encouraging and positive.
