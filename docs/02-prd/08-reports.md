# Product Requirements Document

# 08 - Reports

---

# Overview

The Reports module provides meaningful insights for Users, Mentors, Region Admins, and Super Admins.

Reports should focus on encouraging improvement, measuring consistency, and simplifying follow-up.

Reports should never be used to publicly compare users.

---

# Objectives

The Reports module should:

- Measure consistency.
- Display progress.
- Help mentors follow users.
- Provide actionable insights.
- Generate weekly summaries.
- Support decision making.

---

# Report Types

The application supports:

- Daily Reports
- Weekly Reports
- Monthly Reports
- User Reports
- Mentor Reports
- Region Reports
- System Reports

---

# User Report

Each user has a personal report.

It includes:

- Current Worship Level
- Gamification Level
- Total XP
- Current Streak
- Longest Streak
- Consistency Percentage
- Weekly Completion Rate
- Monthly Completion Rate
- Total Worship Score
- Earned Badges
- Achievements
- Weekly Missions
- Journey Timeline

---

# Mentor Dashboard

Mentors should see:

- Assigned Users
- Active Users
- Inactive Users
- Users Missing Tracking Today
- Users Ready For Promotion
- Average Consistency
- Weekly Reviews Pending
- Recent Activities

---

# Region Dashboard

Region Admins should see:

- Total Mentors
- Total Users
- Active Users
- Inactive Users
- Weekly Completion Rate
- Average Consistency
- Average Streak
- Promotions This Month
- Most Active Mentors

---

# Super Admin Dashboard

The Super Admin dashboard displays:

- Total Regions
- Total Mentors
- Total Users
- Active Users
- New Registrations
- Daily Tracking Rate
- Average Consistency
- Promotion Statistics
- System Health
- Email Delivery Status

---

# Charts

Supported charts:

- Daily Activity
- Weekly Activity
- Monthly Activity
- Worship Distribution
- Consistency Trend
- XP Growth
- Streak History
- Promotion Timeline

Charts should support:

- Weekly View
- Monthly View
- Yearly View

---

# Weekly Summary

At the end of every week the system generates:

For Users:

- Worship Completion
- XP Earned
- Consistency
- Achievements
- Mentor Feedback

For Mentors:

- User Activity
- Missing Users
- Promotion Suggestions

---

# Monthly Summary

Monthly reports include:

- Monthly Consistency
- Monthly XP
- Completed Missions
- Best Week
- Weakest Week
- Worship Breakdown

---

# Journey Timeline

Every user has a timeline.

Example Events:

- Joined THABAT
- Assigned Mentor
- Level Promotion
- Achievement Unlocked
- Badge Earned
- Mission Completed

The timeline should help users visualize their progress.

---

# Export

Reports can be exported as:

- PDF
- Excel

Future support:

- CSV

---

# Filters

Reports should support filtering by:

- Date Range
- Region
- Mentor
- Worship Level
- User Status

---

# Permissions

Users:

View only their own reports.

Mentors:

View reports of assigned users only.

Region Admins:

View reports for their region only.

Super Admins:

View all reports.

---

# Performance Indicators (KPIs)

The application calculates:

- Daily Tracking Rate
- Weekly Completion Rate
- Monthly Completion Rate
- Consistency Percentage
- Average XP
- Average Streak
- Promotion Rate
- Active User Percentage

---

# Acceptance Criteria

✓ User reports available.

✓ Mentor dashboard available.

✓ Region dashboard available.

✓ Super Admin dashboard available.

✓ Charts supported.

✓ Weekly summaries generated.

✓ Monthly summaries generated.

✓ PDF export supported.

✓ Permission-based access enforced.
