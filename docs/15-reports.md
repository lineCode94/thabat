# Notification System

Version: 1.0

Status: Approved

---

# Overview

The notification system helps users stay consistent without becoming distracting.

Notifications should encourage, remind, and celebrate progress.

Notifications should never shame users.

---

# Notification Channels

Version 1

- In-App Notifications
- Email Notifications

Future Versions

- WhatsApp
- Push Notifications (FCM / APNs)
- SMS

---

# Notification Types

## Reminder

Examples

- Don't forget today's tracking.
- Complete your Quran reading.
- Weekly review is available.

---

## Achievement

Examples

- Badge unlocked.
- Achievement earned.
- New title unlocked.

---

## Mentor

Examples

- New mentor comment.
- Weekly review completed.
- Promotion approved.
- New mission assigned.

---

## Administrative

Examples

- Account updated.
- Password changed.
- Region transferred.
- Worship level changed.

---

# Daily Reminder Rules

Send reminder only if:

- User has not completed today's tracking.
- Current time is after the configured reminder time.

Only one reminder per day.

---

# Weekly Reminder

Sent automatically after week closing.

Includes:

- Weekly consistency
- Weekly XP
- Mentor review (if available)

---

# Mission Notifications

Notify when:

- Mission assigned
- Mission almost expires
- Mission completed

---

# Badge Notifications

Trigger immediately after unlocking.

Display:

- Badge Icon
- Badge Name
- Short congratulatory message

---

# Achievement Notifications

Trigger immediately after completion.

Celebrate with subtle animation.

---

# Mentor Notifications

Notify mentors when:

- User missed several days
- Weekly review is pending
- User is eligible for promotion
- New user assigned

---

# Admin Notifications

Notify admins when:

- New mentor created
- Region statistics available
- Weekly reports generated
- System errors detected

---

# Notification Settings

Users can enable or disable:

- Email notifications
- Daily reminders
- Weekly reminders
- Achievement notifications
- Mission notifications

Critical system notifications cannot be disabled.

---

# Notification Status

Every notification has:

- Unread
- Read
- Archived

Users can mark individual notifications or all as read.

---

# Notification Priority

High

Medium

Low

High priority notifications appear first.

---

# Notification Retention

Notifications are stored for historical reference.

Old notifications may be archived automatically.
