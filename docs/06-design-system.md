# Design System

Version: 1.0

Status: Approved

---

# Overview

The THABAT Design System ensures a consistent visual language across the application.

Every page, component, and interaction should follow the same design rules.

The design should be modern, clean, calming, and easy to use.

---

# Design Principles

- Consistency
- Simplicity
- Accessibility
- Clarity
- Performance
- Reusability

---

# Theme Support

The application supports:

- Light Mode
- Dark Mode

The selected theme should be saved for each user.

# Design Rules

- Use consistent spacing.
- Keep layouts clean.
- Prefer cards over dense tables.
- Minimize clicks.
- Highlight progress.
- Encourage users through visuals.
- Avoid visual clutter.
