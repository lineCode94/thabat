# Gamification System

Version: 1.0

Status: Approved

---

# Overview

The purpose of gamification is to encourage consistency, not competition.

The application should motivate users through positive reinforcement.

Users should never feel embarrassed or compared publicly.

There are no public leaderboards.

The focus is on personal growth.

---

# Core Principles

- Encourage consistency.
- Celebrate achievements.
- Reward effort.
- Avoid unhealthy competition.
- Keep users motivated.
- Make progress visible.

---

# Player Profile

Every user has:

- Worship Level
- Gamification Level
- XP
- Current Streak
- Longest Streak
- Consistency Score
- Earned Badges
- Achievements
- Completed Missions

---

# XP System

XP is earned by completing worship activities.

Every worship item has a fixed XP value.

Example

Fajr

20 XP

Morning Azkar

10 XP

Quran

2 XP per Page

Lesson Attendance

30 XP

Book Reading

40 XP

XP is never deducted.

---

# Level System

Gamification levels are independent from Worship Levels.

XP-based Gamification Levels must not be used to approve or infer Worship Level promotion.

Example

Level 1

0 XP

Level 2

500 XP

Level 3

1200 XP

Level 4

2200 XP

Level 5

3500 XP

XP requirements should be configurable.

---

# Worship Levels

Promotion between worship levels is never automatic.

Only mentors can promote users.

The system only provides recommendations.

Worship Level recommendations use configurable readiness criteria and always require authorized human approval.

---

# Streak

Current Streak

Number of consecutive active days.

Longest Streak

Highest streak achieved.

Breaking a streak resets only the current streak.

---

# Consistency

Consistency is calculated separately from XP.

Formula

Completed Score

÷

Assigned Score

×

100

Displayed as a percentage.
